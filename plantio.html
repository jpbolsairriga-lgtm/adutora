Como configurar a Chave API do Google Maps

Atenção: O Google oferece US$ 200 de crédito gratuito por mês — suficiente para uso profissional sem custo.
1

Acessar o Google Cloud Console

Acesse com sua conta Google e crie um projeto.

↗ Abrir Console
2

Ativar faturamento

Vá em Faturamento e vincule um cartão. Não será cobrado dentro do crédito mensal.

3

Ativar as APIs

✓ Maps JavaScript API
✓ Maps Elevation API
↗ Biblioteca de APIs
4

Criar e restringir a chave

APIs & Serviços → Credenciais → Criar Chave. Restrinja selecionando as APIs acima.
