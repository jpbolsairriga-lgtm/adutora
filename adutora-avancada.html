Como configurar a Chave API do Google Maps

Atenção: A chave API é pessoal. O Google oferece US$ 200 de crédito gratuito por mês — suficiente para uso profissional sem custo.
1

Acessar o Google Cloud Console

Acesse com sua conta Google. Crie um projeto se ainda não tiver.

↗ Abrir Console
2

Ativar faturamento (obrigatório, mas gratuito)

Vá em Faturamento e vincule um cartão. Não será cobrado dentro do crédito mensal.

3

Ativar as 3 APIs

✓ Maps JavaScript API
✓ Maps Elevation API
✓ Maps Static API (para o relatório PDF)
↗ Biblioteca de APIs
4

Criar e restringir a chave

APIs & Serviços → Credenciais → Criar Chave. Depois restrinja: selecione as 3 APIs acima.

↗ Credenciais
5

Colar a chave no aplicativo

Cole no campo Chave API Google e clique OK. Fica salva no navegador.
